# Результаты и артефакты

Папка `results/` в рабочем режиме содержит:

- run-папки отдельных запусков;
- `checkpoint_best.pt`;
- `history.json`;
- `metrics.json`;
- графики обучения;
- дополнительные вспомогательные файлы viewer-а.

## Что хранится в Git

В Git версионируются только **лёгкие и полезные итоговые сводки**, а не тяжёлые бинарные артефакты.

Сейчас в Git оставлен:

- `drive_real_comparison.md` — краткая сводка основных реальных результатов на DRIVE.

## Что не хранится в Git

В Git не должны попадать:

- чекпоинты `*.pt`;
- полные run-папки;
- тяжёлые графики и временные артефакты;
- промежуточные эксперименты большого объёма.

## Зачем так сделано

Это позволяет одновременно:

- сохранить воспроизводимость проекта;
- не раздувать репозиторий;
- оставить в истории Git самые важные числовые результаты.
