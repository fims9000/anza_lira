# Results Directory

Эта папка хранит только рабочие и финальные материалы для статей.

## Что считается финальным

- `*_final_*_pack*.md`
- `*_revision_materials*.md`
- `*_metrics_*_pack*.md`
- директории с итоговыми figure assets, которые используются в тексте статьи.

## Что считается промежуточным

- быстрые пробы (`quick`, `probe`, `debug`, `smoke`);
- временные очереди запусков;
- локальные абляции без итоговой фиксации протокола.

Промежуточные результаты либо архивируются, либо не попадают в итоговые таблицы.

## Текущее состояние по статьям

1. **Статья 2 (EDM)** — в активной сборке (метрики + фигуры + ответы рецензентам).
2. **Статья 1 (IMSC)** — текстовые целевые метрики оставлены как рабочие ориентиры до окончательной фиксации.
3. **Статья 3** — следующий этап, после стабилизации статьи 2.
