## 2) Явная легенда (для новой версии рисунка)

- `Error map` (для baseline и AZ): зелёный = TP, красный = FP, синий = FN, чёрный = TN.
- `Difference map (AZ vs Baseline)`:
  - зелёный = AZ прав, baseline ошибся
  - красный = baseline прав, AZ ошибся
  - белый = оба верно на пикселях дороги
  - чёрный = оба ошиблись
  - тёмно-серый = оба верно на фоне
- `Skeleton improvement map`:
  - зелёный = AZ восстанавливает GT-скелет, где baseline пропустил
  - красный = baseline лучше AZ по GT-скелету
  - белый = оба попали в GT-скелет
  - синий = оба пропустили GT-скелет
- Примечание по clDice: на этом наборе и при этих порогах clDice-семейство почти не различает baseline/AZ, поэтому основной акцент для road-cases делаем на Dice/IoU/Precision/Recall и связности.
